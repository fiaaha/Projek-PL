Panduan Anda untuk Menata Web dengan Indah

Bab ini akan menjadi gerbang awal Anda untuk memahami salah satu teknologi inti dalam pengembangan web. Kita akan mengupas tuntas apa itu CSS, perannya, dan bagaimana cara kerjanya dalam mempercantik tampilan sebuah situs web.

Apa itu CSS?

Pengertian CSS dan fungsinya dalam web

CSS adalah singkatan dari Cascading Style Sheets. Secara sederhana, CSS adalah bahasa yang digunakan untuk mendeskripsikan dan mengatur tampilan visual dari sebuah halaman web. Jika HTML (HyperText Markup Language) bertugas membangun struktur atau kerangka sebuah halaman—seperti menentukan mana yang menjadi judul, paragraf, atau gambar—maka CSS-lah yang memberikan gaya pada struktur tersebut.

Fungsi utama CSS meliputi:

  • Tata Letak (Layout): Mengatur posisi elemen-elemen di halaman, seperti membuat kolom, header, footer, dan sidebar.
  • Pengaturan Warna dan Font: Dengan CSS, kita bisa mengubah warna teks, latar belakang, dan jenis font yang digunakan.
  • Responsif: CSS mendukung desain responsif, yang memungkinkan halaman web terlihat baik di berbagai perangkat, dari desktop hingga smartphone.

Dengan CSS, Anda bisa mengubah situs web yang tadinya hanya berupa teks hitam-putih menjadi sebuah karya visual yang menarik dan mudah digunakan.

Peran CSS dalam arsitektur front-end

Dalam pengembangan front-end, CSS berperan sebagai layer presentasi, yaitu bagian yang mengatur bagaimana konten ditampilkan kepada pengguna. CSS bekerja bersama HTML (struktur konten) dan JavaScript (interaktivitas). Dengan CSS, pengembang dapat menciptakan pengalaman pengguna (UI/UX) yang lebih responsif dan modern.

Dalam arsitektur front-end (bagian dari aplikasi web yang berinteraksi langsung dengan pengguna), terdapat tiga pilar utama:

  • HTML (HyperText Markup Language): Menyediakan struktur dasar dari halaman web, seperti teks, gambar, dan tautan.
  • CSS (Cascading Style Sheets): Mengatur gaya visual dari elemen-elemen HTML, seperti warna, ukuran font, dan tata letak.
  • JavaScript: Menambahkan interaktivitas dan dinamika pada halaman web, seperti animasi, validasi form, dan manipulasi DOM.

Peran CSS sangat krusial karena ia memisahkan antara konten (HTML) dengan presentasi (CSS). Pemisahan ini membuat kode lebih rapi, mudah dikelola, dan efisien. Seorang pengembang dapat mengubah seluruh tampilan situs hanya dengan mengubah file CSS, tanpa perlu menyentuh struktur HTML-nya sama sekali.

Perbandingan CSS dengan HTML

Untuk memahaminya lebih mudah, mari kita gunakan analogi membangun rumah:

Aspek HTML CSS
Fungsi Membangun struktur rumah (dinding, atap, pintu) Menghias rumah (cat dinding, memilih furnitur)
Contoh Tag HTML seperti <h1>, <p>, <img> Aturan CSS seperti color, font-size, margin
Interaksi Menentukan apa yang ada di halaman Menentukan bagaimana tampilannya

Singkatnya, HTML adalah tentang apa kontennya, sedangkan CSS adalah tentang bagaimana tampilan kontennya. Keduanya bekerja sama untuk menciptakan halaman web yang fungsional dan menarik secara visual.

Struktur dan Sintaks CSS

Struktur penulisan: selector, property, value

Setiap aturan CSS pada dasarnya terdiri dari dua bagian utama: selector dan declaration block.


selector { 
    property: value; 
}
  • Selector: Bagian ini menentukan elemen HTML mana yang akan diberi gaya. Misalnya, jika Anda ingin mengubah warna teks pada semua paragraf, Anda akan menggunakan selector <p>.
  • Property: Ini adalah atribut yang ingin Anda ubah. Contohnya, 'color' untuk mengubah warna teks atau 'font-size' untuk mengubah ukuran font.
  • Value: Nilai yang diberikan pada properti tersebut. Misalnya, 'red' untuk warna merah atau '16px' untuk ukuran font 16 piksel.
    • Selector: Bagian ini menentukan elemen HTML mana yang akan diberi gaya. Misalnya, jika Anda ingin mengubah warna teks pada semua paragraf, Anda akan menggunakan selector <p>.

Cara Menulis CSS

Tiga Cara Penulisan CSS

Inline CSS: definisi, kelebihan & kekurangan

Inline CSS ditulis langsung di dalam tag HTML menggunakan atribut style. Metode ini hanya akan memengaruhi elemen spesifik di mana ia diterapkan.

  • Definsi:Aturan CSS ditempatkan di dalam atribut style dari sebuah elemen HTML.
  • Kelebihan:
    • Cepat untuk pengujian: Sangat berguna untuk menguji atau menerapkan gaya unik pada satu elemen dengan cepat.
    • Prioritas tertinggi: Aturan inline memiliki spesifisitas tertinggi, artinya akan menimpa aturan dari internal maupun external stylesheet.
  • Kekurangan:
    • Sulit dikelola: Menjadikan pemeliharaan situs web yang besar menjadi sulit karena perubahan harus dilakukan pada setiap elemen satu per satu.
    • Mencampur struktur dan presentasi: Mengaburkan pemisahan antara konten (HTML) dan desain (CSS), yang dianggap sebagai praktik yang kurang baik.
    • Tidak dapat digunakan kembali: Gaya tidak dapat diterapkan kembali ke elemen lain tanpa menyalin kode yang sama berulang kali.

Contoh penggunaan inline CSS:

<p style="color: red; font-size: 20px;">Ini adalah paragraf dengan inline CSS.</p>

Internal CSS: penggunaan dalam satu file HTML

Internal CSS (atau embedded CSS) didefinisikan di dalam tag style yang ditempatkan di bagian head dari sebuah file HTML. Metode ini memengaruhi semua elemen yang sesuai di dalam satu halaman tersebut.

  • Penggunaan dalam satu file HTML: Aturan CSS ditulis di dalam tag <style> yang berada di dalam bagian <head> dari dokumen HTML. Gaya yang didefinisikan di sini hanya berlaku untuk halaman tersebut.
  • Kelebihan:
    • Berguna untuk gaya per halaman:
      • Ideal ketika sebuah halaman tunggal memerlukan gaya yang unik dan tidak akan digunakan di halaman lain.
      • Meningkatkan keterbacaan kode dengan memisahkan gaya dari struktur HTML.
  • Kekurangan:
    • Hanya berlaku untuk satu halaman: Tidak efisien jika gaya yang sama perlu diterapkan di beberapa halaman, karena kode CSS harus disalin ke setiap halaman.
    • Memperbesar ukuran file HTML: Menambah ukuran file HTML, yang dapat sedikit memengaruhi waktu muat halaman.

Contoh penggunaan internal CSS:


<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: lightgray;
    }
    h1 {
      color: navy;
    }
  </style>
</head>
<body>
  <h1>Judul Ini Berwarna Navy</h1>
</body>
</html>

External CSS (Fokus utama): penulisan file .css, cara link di HTML

External CSS adalah metode di mana aturan CSS ditulis dalam file terpisah dengan ekstensi .css. File ini kemudian dihubungkan ke file HTML menggunakan tag link di bagian head.

  • Penulisan file .css: Anda membuat sebuah file baru (misalnya, style.css) dan menulis semua aturan CSS Anda di dalamnya tanpa tag HTML apa pun.

Contoh penulisan file style.css:


body { 
  font-family: Arial, sans-serif; 
  background-color: #f4f4f4; 
} 

h1 { 
  color: #333; 
}
  • Cara link di HTML: File CSS eksternal dihubungkan ke dalam dokumen HTML menggunakan tag <link> di dalam bagian <head>. Atribut rel="stylesheet" memberi tahu browser bahwa ini adalah file stylesheet, dan href menentukan lokasi file .css tersebut.
